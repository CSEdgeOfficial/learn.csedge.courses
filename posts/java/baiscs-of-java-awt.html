Mastering AWT Java: A to Z

Java AWT (Abstract Window Toolkit) is an API to develop GUI or window-based applications in java. Java AWT components are platform-dependent i.e. components are displayed according to the view of operating system. AWT is heavyweight i.e. its components are using the resources of OS. The AWT is part of the Java Foundation Classes (JFC) â€” the standard API for providing a graphical user interface (GUI) for a Java program. The java.awt package provides classes for AWT api such as TextField, Label, TextArea,RadioButton, CheckBox, Choice, List etc. AWT is also the GUI toolkit for a number of Java ME profiles. For example, Connected Device Configuration profiles require Java runtimes on mobile telephones to support the Abstract Window Toolkit.

Java AWT Hierarchy


AWT Components

  • Button: A clickable button.
    • button = new Button("Click Me");
  • Label: A non-editable text field.
    • Label label = new Label("This is a label");
  • TextField: A single-line text input field.
    • TextField textField = new TextField(20);
  • TextArea: A multi-line text input field.
    • TextArea textArea = new TextArea(5, 20);
  • Checkbox: A checkbox that can be selected or deselected.
    • Checkbox checkbox = new Checkbox("Accept Terms");

AWT containers

Containers are used to hold and organize components. Key containers include:

  • Frame: A top-level window with a title and border. 
    •    
Frame frame = new Frame("My Application");
frame.setSize(400, 300);
frame.setVisible(true);
  • Panel: A simple container that can group components together.
    • Panel panel = new Panel();

AWT Event handling

Event handling is crucial for creating interactive applications. Events are handled by implementing listener interfaces.

  • ActionListener: For handling actions like button clicks.
    • 
button.addActionListener(new ActionListener() {
  public void actionPerformed(ActionEvent e) {
    // Handle button click
  }
});
  • MouseListener: For handling mouse events.
    • 
canvas.addMouseListener(new MouseAdapter() {
  public void mouseClicked(MouseEvent e) {
    // Handle mouse click
  }
});

AWT Layout Managers

Layout managers control the positioning and sizing of components within a container.

  • FlowLayout: Arranges components in a left-to-right flow.
    • panel.setLayout(new FlowLayout());
  • BorderLayout: Divides the container into five regions (North, South, East, West, Center).
    • frame.setLayout(new BorderLayout());
