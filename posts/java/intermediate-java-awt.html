Intermediate level

In the intermediate level of our guide, we dive deeper into the intricacies of Java GUI development. Building upon the foundational knowledge acquired in the basic level, this section introduces more advanced components, event handling techniques, and layout management strategies. Here, you will explore how to create custom graphics, handle window events effectively, and employ sophisticated layout managers to design complex user interfaces. By mastering these intermediate-level concepts, you will be equipped with the skills needed to develop more interactive and visually appealing Java applications.

Advanced Components

More advanced AWT components offer additional functionality.

  • Choice: A dropdown menu. 
                                
choice choice = new Choice();
choice.add("Option 1");
choice.add("Option 2");

  • List: A list of items.

    • List list = new List(4, false);
list.add("Item 1");
list.add("Item 2");
  • Scrollbar: A scrollbar component.

Window Events

Window events handle actions on windows, such as closing or resizing.

  • WindowAdapter: An adapter class for receiving window events
    • 
frame.addWindowListener(new WindowAdapter() {
    public void windowClosing(WindowEvent e) {
        System.exit(0);
    }
});

Custom Graphics

Custom graphics can be drawn using the Graphics class.

  • Drawing Shapes: Using methods like drawLine(), drawRect(), and drawOval().
    • 
public void paint(Graphics g) {
    g.drawLine(10, 10, 50, 50);
    g.drawRect(70, 10, 50, 50);
    g.drawOval(130, 10, 50, 50);
}

Layout Managers

More complex layout managers offer greater control over component placement.

  • GridLayout: Arranges components in a grid of equal-sized cells. 
    panel.setLayout(new GridLayout(2, 3));
  • GridBagLayout: A flexible layout manager for complex layouts. 
    panel.setLayout(new GridBagLayout());
