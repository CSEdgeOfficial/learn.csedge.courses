First Java Program

Introduction

In this article, we will learn how to write and run a simple "Hello World" program in Java. This program is often the first example taught to beginners in Java programming as it demonstrates the basic structure and syntax of a Java program.

Prerequisites

To follow along with this tutorial, you need:

  • A computer with a text editor installed
  • Java Development Kit (JDK) installed

Step 1: Writing the Java Code

Open your favorite text editor and create a new file named HelloWorld.java.

Now, add the following Java code to the file:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}

This Java code defines a class named HelloWorld with a main method. Inside the main method, it prints "Hello, world!" to the console using System.out.println().

Step 2: Compiling the Java Code

Now that we have written our Java code, we need to compile it into bytecode that can be executed by the Java Virtual Machine (JVM).

Open a terminal or command prompt, navigate to the directory containing HelloWorld.java, and run the following command:

javac HelloWorld.java

If there are no syntax errors in your code, this command will generate a file named HelloWorld.class.

Step 3: Running the Java Program

Now that we have compiled our Java code, we can run it using the java command.

In the same terminal or command prompt, run the following command:

java HelloWorld

This command will execute the HelloWorld class, and you should see the output Hello, world! printed to the console.

Conclusion

Congratulations! You have successfully written and run a "Hello World" program in Java. This simple program serves as a foundation for learning more advanced concepts in Java programming.
